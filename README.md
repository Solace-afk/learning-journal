# Jurnal Belajar

Repositori ini berfungsi sebagai arsip pembelajaran pribadi untuk mata kuliah *Reverse Engineering*. Setiap direktori didesain layaknya jurnal mingguan yang mencatat progres belajar, pemahaman teori, hingga tantangan yang ditemukan selama proses pembedahan sistem keamanan.

Fokus utama dari dokumentasi ini adalah untuk melacak perkembangan pembelajaran saya.

---

## Cakupan Jurnal

Di dalam setiap jurnal mingguan, Anda akan menemukan dokumentasi berupa:
- Intisari dari materi atau teori yang dipelajari pada minggu tersebut.
- Penjelasan konsep-konsep teknis menggunakan pemahaman saya sendiri.
- Hambatan, batasan, atau konsep yang masih belum saya pahami.

## Capaian yang Diharapkan

Secara garis besar, repositori ini dibangun dengan target agar saya dapat:
- Menguasai dasar-dasar pembedahan file *binary*.
- Mampu membedakan serta menerapkan pendekatan *Static Analysis* dan *Dynamic Analysis*.
- Mengoperasikan perangkat lunak *decompiler* maupun *disassembler*.
- Mengasah ketelitian dan logika pemecahan masalah melalui penyelesaian tantangan *Crackme*.
- Memiliki rekam jejak dokumentasi yang rapi untuk referensi di masa depan.
